# Exam Prep

A full-stack exam preparation web application built with:

* Next.js App Router
* TypeScript
* Tailwind CSS
* Prisma
* SQLite
* shadcn/ui

## Features

* User authentication
* Question paper creation
* MCQ test taking
* Result analysis
* Negative marking support
* Reattempt tests
* Dashboard with sorting and grouping
* Responsive UI

## Tech Stack

* Next.js 14
* Prisma ORM
* SQLite
* Tailwind CSS
* shadcn/ui
* Lucide React

## Run Locally

```bash
npm install
npx prisma generate
npx prisma migrate dev
npm run dev
```

## Build Production

```bash
npm run build
npm start
```
